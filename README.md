# @pankaj15/simple-addition

Tiny npm package for adding, subtracting, and multiplying two numbers.

## Install

```bash
npm install @pankaj15/simple-addition
```

## Usage

```js
const { add, subtract, multiply } = require('@pankaj15/simple-addition');

console.log(add(2, 3)); // 5
console.log(subtract(10, 4)); // 6
console.log(multiply(3, 4)); // 12
```
